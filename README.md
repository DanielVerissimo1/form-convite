**Festivite** é uma aplicação desenvolvida durante os estudos na **Rocketseat**, com o objetivo de permitir a criação de convites digitais personalizados para diferentes tipos de eventos.

O usuário pode preencher informações essenciais do evento, escolher estilos visuais e gerar um convite moderno e atrativo.

---

## 🚀 Funcionalidades:

- Criação de convites digitais personalizados  
- Preenchimento de informações do evento:
  - Título  
  - Data  
  - Local  
  - Descrição  
- Personalização visual:
  - Escolha de cores  
  - Temas  
  - Imagens  
- Adição de informações de contato  
- Aceite de termos antes da geração  
- Interface intuitiva e organizada  

---

## 🛠️ Tecnologias utilizadas:

- HTML5  
- CSS3
